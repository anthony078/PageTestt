# 🔥 Proyecto desarrollado para el curso de **Interacción Humano Computador**.

Plataforma web innovadora orientada a la venta de productos deportivos, diseñada para mejorar la experiencia de compra mediante un probador virtual interactivo, catálogo dinámico de productos y herramientas de análisis para la toma de decisiones.

## 📌 Descripción

 Es una solución web desarrollada como proyecto académico que busca optimizar el proceso de compra de prendas y accesorios deportivos. La plataforma permite a los usuarios explorar productos, visualizar combinaciones mediante un probador virtual y acceder a una experiencia moderna, intuitiva y responsive.

## 🚀 Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Responsive Design
- Git & GitHub

## ✨ Funcionalidades Principales

- 👕 Catálogo interactivo de productos deportivos.
- 🧍 Probador virtual para visualizar prendas y accesorios.
- 🛒 Carrito de compras.
- 📊 Dashboard de análisis y estadísticas.
- 🔍 visualización de productos.
- 📱 Diseño adaptable a dispositivos móviles, tablets y escritorio.
- 🎨 Interfaz moderna enfocada en la experiencia de usuario (UX/UI).

## 🎯 Objetivo del Proyecto

Desarrollar una plataforma web que mejore la experiencia de compra de productos deportivos mediante herramientas interactivas que faciliten la visualización, selección y gestión de productos.
